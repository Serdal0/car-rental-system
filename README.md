# Car Rental System

Full-stack araç kiralama sistemi.

## Project Overview
Bu proje, araç kiralama süreçlerini yönetmek için geliştirilmiş full-stack bir uygulamadır.
Backend tarafı Spring Boot ile RESTful API olarak tasarlanmış, frontend tarafı kullanıcı arayüzünü sağlar.

## Tech Stack
